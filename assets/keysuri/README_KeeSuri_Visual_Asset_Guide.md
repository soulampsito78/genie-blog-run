## Kee-Suri Identity

Kee-Suri is a **glamorous premium AI tech secretary**.

She is **not**:

- Genie
- A news anchor
- A CEO or executive persona

Every generated image must read as Kee-Suri: polished, intelligent, approachable, and distinctly separate from other personas in this project.

## Generation Rules

1. **One image per generation prompt only** — attach or reference a single reference asset per prompt, not both at once unless running separate generations.
2. **No collage** — do not combine multiple reference images into one output frame.
3. **No split screen** — do not produce side-by-side or divided layouts.
4. **No two images in one frame** — each output must be a single coherent scene with one subject.

When in doubt, pick the asset that best matches the shot type (upper-body briefing → 01; full-body → 02) and generate again rather than merging references.
