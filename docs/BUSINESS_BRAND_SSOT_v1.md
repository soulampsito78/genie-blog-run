# 프라이빗 브리핑 — 사업·브랜드 SSOT v1

**상태:** APPROVED  
**버전:** v1  
**작성일:** 2026-06-23  
**문서 권한:** 이 문서는 운영자, 투자자, 협력사, 마케팅 자료가 참조하는 사업·브랜드 단일 진실 공급원(SSOT)이다.  
**코드 상태와의 관계:** 기술 구현 세부 사항은 `docs/CURRENT_STATUS_SNAPSHOT.md`를 참조. 이 문서는 사업·브랜드 정의를 다루며 코드 구현 상태와 독립적으로 유지된다.

---

## 1. 서비스 정의

**상위 서비스명:** 프라이빗 브리핑

> 평일마다 시장과 기술 신호를 AI가 구조화하고,  
> 운영자가 검수한 브리핑만 이메일로 전달하는  
> **검수형 AI 브리핑 서비스.**

이 서비스는 뉴스 요약 서비스가 아니다.  
AI가 초안을 작성하고, 운영자 검수를 통과한 브리핑만 고객에게 전달한다.  
없는 수치를 만들지 않고, 근거 없는 신호는 발송하지 않는다.

---

## 2. 운영 원칙

```
No Hallucinations.
Fact-Based.
Human-Verified.
```

**한국어 표현:**

> 환각 없이, 사실 기반으로, 사람이 검수한 브리핑만.

**원칙별 의미:**

| 원칙 | 의미 |
|------|------|
| No Hallucinations | AI가 생성한 수치·정보 중 근거가 없는 것은 발송하지 않는다 |
| Fact-Based | 시장 데이터, 뉴스 출처, 기술 신호에 기반한 내용만 포함한다 |
| Human-Verified | 운영자 검수 및 승인 절차를 거친 브리핑만 고객에게 전달된다 |

## 4. 서비스별 설명

### 4.1 오늘의 지니 (Today_Geenee)

평일 장전, 오늘 시장에서 먼저 확인해야 할 변수와 리스크 신호를 정리하는 시장 브리핑.

| 항목 | 내용 |
|------|------|
| 발행 주기 | 평일 (월–금) |
| 발행 시각 | 장전 오전 |
| 콘텐츠 도메인 | 시장·금융·매크로·리스크 |
| 포맷 | 이메일 브리핑 |

### 4.2 키수리 글로벌 (Key-Suri Global)

평일, 글로벌 AI·빅테크·반도체·플랫폼 신호를 정리하는 프라이빗 테크 브리핑.

| 항목 | 내용 |
|------|------|
| 발행 주기 | 평일 (월–금) |
| 발행 시각 | 낮 |
| 콘텐츠 도메인 | 글로벌 AI, 빅테크, 반도체, 플랫폼, 정책 |
| 포맷 | 이메일 브리핑 |

### 4.3 키수리 국내 (Key-Suri Korea)

평일, 국내 AI·스타트업·플랫폼·정책 변화를 정리하는 프라이빗 테크 브리핑.

| 항목 | 내용 |
|------|------|
| 발행 주기 | 평일 (월–금) |
| 발행 시각 | 저녁 |
| 콘텐츠 도메인 | 국내 AI, 스타트업, 플랫폼, 정책, 지원사업 |
| 포맷 | 이메일 브리핑 |

### 4.4 내일의 지니 (Tomorrow_Geenee)

**상태: RETIRED**

서비스 포트폴리오에서 제외. 외부 소개·마케팅 자료에 포함하지 않는다.

## 9. 금지 표현

외부 소개, 마케팅 자료, 이메일, 랜딩 페이지에서 아래 표현을 사용하지 않는다.

| 금지 표현 | 이유 |
|----------|------|
| 투자 추천 | 금융 규제 및 브랜드 원칙 위반 |
| 매수 추천 | 동일 |
| 매도 추천 | 동일 |
| 수익 보장 | 사실 기반 원칙 위반 |
| 실시간 뉴스 | 브리핑은 구조화된 신호 정리이며 실시간 뉴스피드가 아님 |
| 완전 자동 발행 | 운영자 검수 원칙 위반 |
| 무인 운영 | 동일 |
| 365일 브리핑 | 평일 발행 원칙 위반 |
| 주말 포함 발송 | 동일 |

## 10. 서비스 운영 구조 요약 (비기술 설명)

고객이 볼 필요 없는 기술 세부 사항은 생략. 운영 방식 이해를 위한 최소 요약만 포함.

1. AI가 매일 평일 시장·기술 신호를 기반으로 브리핑 초안을 생성한다.
2. 운영자가 이메일로 초안을 검토한다.
3. 운영자가 승인한 브리핑만 고객에게 발송된다.
4. 승인되지 않은 브리핑은 발송되지 않는다.
5. 주말에는 발송하지 않는다.
