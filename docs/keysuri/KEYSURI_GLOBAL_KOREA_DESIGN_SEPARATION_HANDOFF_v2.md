# Kee-Suri Global / Korea Briefing Design Separation Handoff (v2 — Global 주간 톤 보정)

> 디자인 핸드오프 전용. 프로덕션 코드·백엔드 로직·이미지 생성·이메일 발송·스케줄러 변경·신규 기능 없음.
> v1 대비 변경점: **편집 분리는 유지**, **Global 비주얼 시스템을 잠금 이미지 프로필에 맞춰 '밝은 낮' 톤으로 전면 보정**(나이트블루 커맨드룸 폐기). Korea는 저녁·웜·골드 무드 유지.
> 잠금 자산 근거: `KEYSURI_IMAGE_PROFILE_LOCK.md`(Global 12:30 **daytime bright-grey** / Korea 18:30 winter after-sunset), `KEYSURI_TITLE_AND_BODY_SECTION_CONTRACT.md`(섹션 라벨·순서 잠금), `KEYSURI_CONTRACT_PREVIEW_RENDERER_DESIGN.md`.

## 1. 핵심 결론

두 브리핑은 같은 키수리 페르소나 아래의 **서로 다른 두 제품**입니다. 색 변형이 아니라 역할·위계·무드·종료 방식이 다릅니다.

- **Global Tech 12:30 — 밝은 낮의 조기 신호 인텔리전스(outside-in).** 밝은 그레이 주광의 프리미엄 한국 임원 테크 오피스. 깨끗한 실버/소프트 블루/라이트 그래파이트. 전략적이되 사이버펑크 아님, 분석적이되 어둡지 않음. "세계에서 무엇이 움직이는가, 한국 도착 전 주인님이 먼저 봐야 할 압력은 무엇인가."
- **Korea Tech 18:30 — 저녁 국내 해석·퇴근 전 운영 리드아웃(inside-Korea).** 겨울 일몰 후, 웜 차콜·뮤트 골드·앰버, 한국 임원 오피스 저녁. 가깝고 사적이며 약간 따뜻함. "오늘 한국에서 무엇이 바뀌었고, 내일 무엇을 준비할까."

분리는 네 축에서 동시에: ① 편집 역할(렌즈), ② 비주얼 무드, ③ 레이아웃 위계, ④ 카피 톤.

**한 줄 기준(보정):**
- Global = **"밝은 낮에 먼저 보는 세계 기술 지형도"**
- Korea = **"저녁에 정리하고 내일을 준비하는 한국 결재 데스크"**

명도(light/dark)가 두 제품을 가르는 1차 신호입니다. **밝음의 세계는 Global, 어둡고 따뜻한 저녁의 세계는 Korea가 소유합니다.**

## 2. 왜 12:30 Global과 18:30 Korea가 달라야 하는가

| 축 | Global 12:30 | Korea 18:30 |
|----|--------------|-------------|
| 시간 정체성 | **밝은 낮·정오 직후**, 하루의 시작 신호 | 일몰 후·퇴근 전, 하루의 마감 해석 |
| 명도 | **라이트(밝은 그레이 주광)** | 다크(따뜻한 저녁) |
| 정보 방향 | outside-in (세계 → 한국으로 올 압력) | inside-Korea (글로벌·국내 신호 → 한국 적용) |
| 인지 모드 | 탐지·조기 경보 | 해석·실행 준비 |
| 독자 행동 | "도착 전에 알아둬야겠다" | "내일 이걸 어떻게 움직이지" |
| 감정 거리 | 차분하고 전략적, 넓게 | 가깝고 사적, 약간 따뜻 |
| 시각 은유 | **밝은 실버 신호 보드 / 주광 인텔리전스 데스크** | 저녁 결정 데스크 / 정리 메모 |
| 종료 방식 | 다음 48시간 관찰로 **열어둠** | 정리 + 퇴근 메모로 **닫음** |

같은 사건도 역할이 다르므로 결론이 달라야 합니다(§5). 두 제품이 같은 결론을 내면 분리가 무너집니다.

## 3. Global Tech 12:30 디자인 시스템 (밝은 낮 보정)

키워드: **밝은 그레이 글로벌 브리핑 데스크 · 주광 전략 인텔리전스 · 프리미엄 실버 신호 보드 · 깨끗한 12:30 임원 브리핑 · 넓지만 차분한 글로벌 레이어 뷰.**

### 3.1 컬러 팔레트 (라이트 / off-white + 미스트 그레이 + 뮤트 시그널 블루)

| 토큰 | HEX | 용도 |
|------|-----|------|
| `--g-bg` | `#f3f6fa` | 페이지 — 쿨한 off-white 주광 배경 |
| `--g-surface` | `#ffffff` | 카드 — 클린 화이트 |
| `--g-surface-2` | `#eef3f8` | 중첩 블록·출처·분포 바 트랙 |
| `--g-hero-1` → `--g-hero-2` | `#dfe7ef` → `#f8fafc` | 히어로 그라데이션(밝은 미스트→화이트) |
| `--g-line` | `rgba(80,100,130,0.14)` | 헤어라인(라이트) |
| `--g-line-strong` | `rgba(80,100,130,0.24)` | 강조 구분선 |
| `--g-text` | `#172033` | 본문 1차(라이트 그래파이트) |
| `--g-dim` | `#536274` | 2차 |
| `--g-mute` | `#7b8795` | 메타·라벨 |
| `--g-accent` | `#3f7ecb` | 뮤트 시그널 블루 — 구조·레이어·링크 |
| `--g-silver` | `#8fa0b4` | 페일 실버 — 라벨·축·구분 |
| `--g-signal` | `#2ca6a4` | **매우 옅은 시안, '시그널 라인' 한정** — "압력/이동" 포인트만 |

악센트 원칙: **블루 = 글로벌 구조, 실버 = 분석 메타, 시안은 시그널 라인에만 절제 사용.** 골드·앰버·웜톤·다크 네이비 배경 전면 금지(그것은 Korea의 세계).
대비 주의: 라이트 배경이므로 텍스트는 `--g-text #172033` 기준으로 충분한 명도 대비 확보, 옅은 회색 본문 금지.

### 3.2 타이포 무드
- 제목: 단단하고 약간 타이트(`letter-spacing:-0.01em`), 큰 디스플레이 대비. 밝은 배경 위 **그래파이트 텍스트의 또렷한 인텔리전스 헤더**.
- 라벨: 자간 넓은 소형 라벨(`letter-spacing:0.08em`), 실버 — 깨끗한 보드/계기판 톤(어두운 콘솔 아님).
- 본문: 줄간 1.65, 문장 짧고 분석적.

### 3.3 히어로 구성
- 가로로 넓은 **풀-위드 톱샷**(16:9 레터박스, **bright grey overcast daylight window light** — 프로필 잠금 준수).
- 히어로 배경은 **밝은 미스트→화이트 그라데이션**. 그 위에 얇은 **글로벌 레이어 라인**(가로 눈금/위도선 모티프, 실버, 순수 CSS) — 밝은 신호 보드 암시.
- 배지: `글로벌 신호 · 12:30` — **블루-온-라이트 소프트 칩**(`--g-accent` 텍스트 / 옅은 블루 배경).
- 무드: 샤프하되 **밝고 차분한 주광 전략 브리핑**. 야간·커맨드룸·사이버펑크·워룸 금지.

### 3.4 섹션 리듬
넓은 분석 블록 + 큰 여백. 위→아래 **탐지 → 지형 → 압력 → 관찰** 깔때기. 상단에 밝은 신호 보드(분포)를 먼저 둬 "세계가 어디로 가는지"를 첫 화면에 노출.

### 3.5 카드 스타일
- **화이트 카드**(`--g-surface`) + 얇은 라이트 헤어라인 + 아주 옅은 그림자(`0 6px 18px rgba(30,50,80,0.06)`). 밝고 가벼운 보드 카드.
- **블루 좌측 룰**(`border-left:3px solid var(--g-accent)`).
- 랭크: 큰 실버 숫자 + **레이어 태그**(`인프라` `반도체` `로보틱스`).
- 카드 하단 "한국 도착 전 압력" 한 줄을 **시안 시그널 라인**(`--g-signal`, 얇게)으로.

### 3.6 아이콘/라벨 스타일
- 미니멀 라인 아이콘(채움 없음), 실버. 레이어 바(▮▮▮) 등 CSS only.
- 영어 키 노출 금지, 한국어 시그널 라벨만(`레이어 이동` `통제권` `공급망`).

### 3.7 차트/시그널 모티프 (밝은 대시보드)
- **카테고리 분포 바**: `--g-surface-2` 트랙 위 실버/블루 스택 — "오늘 신호가 어느 레이어에 몰렸나". **라이트 대시보드** 룩.
- **레이어 이동 화살표**(↗/→): 블루/실버 텍스트+CSS. 차트 라이브러리 금지. 어두운 콘솔·네온 금지.

### 3.8 이미지 사용 규칙
- 톱샷만 사용(Global은 바텀샷·따뜻한 마무리 없음 — 계약 §11).
- 데이터-URI 임베드(프리뷰), CID 준비(이메일). 실패 시 **밝은 미스트 그라데이션 + 아이덴티티 텍스트**로 graceful fallback(어두운 fallback 금지).

### 3.9 톱샷 비주얼 규칙
- **daytime / bright grey overcast window light**, 태블릿 허용(허리·낮은 가슴 단순 그립), 전략적·안정적 표정.
- 금지: 야간 무드, 저녁/일몰, 아늑한 마감, 오프듀티, 글로벌 커맨드센터·워룸, 앵커/뉴스룸.

## 4. Korea Tech 18:30 디자인 시스템 (유지)

키워드: **한국 저녁 브리핑 · 결정 데스크 · 그라운디드 · 사적·따뜻함.** (v1 유지 — Korea가 어둡고 따뜻한 저녁 세계를 소유.)

### 4.1 컬러 팔레트 (warm / 차콜 + 웜그레이 + 뮤트 골드)

| 토큰 | HEX | 용도 |
|------|-----|------|
| `--k-bg` | `#14110d` | 페이지 — 따뜻한 차콜 |
| `--k-surface` | `#1e1a14` | 카드 |
| `--k-surface-2` | `#27211a` | 중첩·출처 |
| `--k-hero-1` → `--k-hero-2` | `#2a2418` → `#100d0a` | 히어로 그라데이션(저녁 등불) |
| `--k-line` | `rgba(210,190,150,0.16)` | 웜 헤어라인 |
| `--k-text` / `--k-dim` / `--k-mute` | `#f3ece0` / `#c8bca8` / `#8a7d68` | 텍스트 3단 |
| `--k-gold` | `#cda85f` | 시그니처 — 주인님 관점·판단·마무리 |
| `--k-gold-soft` | `rgba(205,168,95,0.14)` | 골드 패널 배경 |
| `--k-ember` | `#b9763f` | 저녁 앰버(절제) — 국내 적용/타이밍 |

악센트 원칙: **골드 = 사적·국내 해석·키수리 판단, 앰버 = 적용/타이밍.** 블루·콜드톤·라이트 배경 금지(그것은 Global의 세계).

### 4.2 타이포 무드
- 제목: 부드럽고 무게감 있게(타이트하지 않음). "퇴근 전 정리 노트".
- 라벨: 자간 보통, 차분. 콘솔 톤 아님.
- 본문: 줄간 1.75, 실용적·구체적.

### 4.3 히어로 구성
- 톱샷 + 저녁 등불 그라데이션. **winter after-sunset, deep blue-gray 도시 야경 + 따뜻한 실내광**(프로필 잠금).
- 글로벌 레이어 라인 금지 — 대신 얇은 저녁 라인/도시 실루엣 힌트 또는 정적 헤더.
- 배지: `국내 해석 · 18:30`(골드-소프트).
- 무드: 가깝고 그라운디드한 저녁. 커맨드센터·주광 금지.

### 4.4 섹션 리듬
해석이 위로. **국내 변화 → 한국 기업·정책 해석 → 내일 영향 → 적용 포인트 → 정리·퇴근 메모.** 마지막은 사적·따뜻하게 닫힘(바텀샷 + 따뜻한 마무리, 프로모션 시).

### 4.5 카드 스타일
- 웜 그라데이션 카드, **골드 좌측 룰**.
- 랭크: 골드 숫자 + **한국 적용 태그**(`국내 적용` `정책` `타이밍`).
- 카드 하단 "내일 영향" 한 줄을 **앰버 라인**으로.

### 4.6 아이콘/라벨 스타일
- 약간 둥근 라인 아이콘, 웜그레이. 데스크/문서/저녁 모티프.
- 한국어 전용 라벨: `국내 해석` `한국 기업` `퇴근 전 메모`.

### 4.7 국내/맥락 모티프
- 글로벌 맵 대신 **"오늘 국내에서 움직인 것" 요약 스트립**(골드/앰버 칩).
- "내일 영향" 미니 타임라인(오늘 → 내일 화살표, 텍스트+CSS).

### 4.8 이미지 사용 규칙
- **톱샷 + 바텀샷** 모두(Korea 18:30 한정, 계약 §11).
- 톱샷=저녁 시작 무드, 바텀샷=오프듀티 마감 무드. 둘 다 데이터-URI/CID, 실패 시 웜 그라데이션 fallback.

### 4.9 톱샷·바텀샷 비주얼 규칙
- **톱샷:** winter after-sunset, 따뜻한 실내광, 태블릿 선택/없음, 손 차분히 모음. 글로벌 커맨드센터·낮 무드 금지.
- **바텀샷(off-duty 마감):** 회장실 문 앞, 퇴근 마무리, **날씨 반영 퇴근룩**, **태블릿·노트북·문서 없음**, "같이 퇴근하는 느낌". 금지: 호텔 라운지, CEO 포트레이트, 야간 유혹 신, 앵커.

## 5. 중복 뉴스 처리 원칙 (유지)

같은 출처/사건은 **각이 다를 때만** 양쪽 등장. 같으면 한쪽에서 제외.

### 5.1 편집 규칙(내부 + 가시)
- **Global 라벨:** `글로벌 원인 / 국제 구조` — 기술 레이어·국제 이동·전략. 한국 함의는 한 줄 압력으로만, 과몰입 금지.
- **Korea 라벨:** `국내 해석 / 한국 적용` — 한국 기업 역할·국내 시장 의미·도메스틱 적용·내일 액션. **글로벌 문장·헤드라인 각·결론 절대 반복 금지.**
- 내부 가드(비가시 메타): 같은 `source_url`이면 `angle: global_cause` / `angle: korea_application` 다르게 강제. 같은 각이면 중복 처리.
- 가시 신호: 중복 사건 카드 상단 칩 — Global `글로벌 원인`(블루-온-라이트) / Korea `국내 적용`(골드).

### 5.2 예시 3종

**① NVIDIA + LG / Doosan**
- Global(`글로벌 원인`): "NVIDIA가 AI 팩토리/피지컬 AI 인프라 전략을 산업 파트너로 확장. AI가 소프트웨어를 넘어 공장·로봇·모빌리티·국가 인프라로 이동하는 구조 신호." → 레이어 이동 초점, 한국 함의는 한 줄 압력.
